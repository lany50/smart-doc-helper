## ✨ 功能特性

- 📸 **普通 OCR 识别** - 支持图片文字提取，识别准确率高
- ✍️ **英语作文批改** - 15 分制智能评分，详细批改报告
- 💡 **写作思路指导** - AI 辅助写作构思，提供框架和范文
- 📱 **移动端友好** - 响应式设计，手机平板均可使用
- 🚀 **多模型支持** - Gemini、Qwen、GPT 等多种 AI 模型可选
