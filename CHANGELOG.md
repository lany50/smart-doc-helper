## [1.0.0] - 2024-01-15

### 新增
- 📸 普通 OCR 识别功能
  - 支持 Gemini Flash、Qwen3-VL、GPT-5 Mini 模型
  - 支持中英文、数字、符号、表格识别
  - 一键复制和下载功能
  
- ✍️ 英语作文批改功能（15分制）
  - 智能三维评分（内容5分 + 语言7分 + 结构3分）
  - 详细批改报告（点评、亮点、问题、建议）
  - 范文参考（约80词）
  - 提分秘诀
  - 支持多图片上传
  
- 💡 写作思路指导
  - 题目分析
  - 写作框架
  - 语言建议
  - 范文示例

### 技术
- 响应式设计，支持移动端
- 支持环境变量配置
- Netlify 部署优化
- 优雅的 UI 设计

### 文档
- 完整的 README
- 详细的部署指南
- 使用说明和示例

**格式说明**：
